![World of Afkcraft](https://raw.githubusercontent.com/80-am/world-of-afkcraft/master/logo.png)

Simple AFK bot for World of Warcraft

## Prerequisites

python3
pyautogui

## Instructions

Checkout repository
```
git clone https://github.com/80-am/world-of-afkcraft.git
```

Start with
```
./world-of-afkcraft/world-of-afkcraft.py
```
